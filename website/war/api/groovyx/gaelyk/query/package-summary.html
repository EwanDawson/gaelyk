Package groovyx.gaelyk.query

 

Class Summary
Clause Base class for sort and filter clauses.
QueryBuilder The query build is used to create a datastore Query or execute a PreparedQuery, from the datastore.query {} or datastore.execute {} calls.
QueryDslTransformation This AST transformation makes two transformations at the AST level.
SortClause A sort clause representing an ordering along a certain entity property.
WhereClause A where clause adds a filtering operation for the query.
 

Enum Summary
QueryType The various options for the select parameter:
  • ALL: select all (full entity, default select option when select omitted)
  • KEYS: select keys (return just the keys)
  • SINGLE: select single (return just one entity when the search yields just one)
  • COUNT: select count (return the count of entities for that query)
 

Exception Summary
QuerySyntaxException Exception thrown when there is a syntax problem in your datastore queries using the datastore query DSL.
