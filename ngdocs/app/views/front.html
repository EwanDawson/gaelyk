Lightweight Groovy Framework for Google App Engine

Start Project

Lazybones

The easiest way to start project is to install Groovy enVironment Manager and let Lazybones create the project for you.

Sign up for Google App Engine and create new application. Use the application id when prompt. 

$ curl -s get.gvmtool.net | bash
$ gvm install lazybones
$ lazybones create gaelyk {{gaelykTemplateVersion}} myproject
$ cd myproject
 Glide

If you prefer simplicity you should try Glide. Glide removes all the clutter needed by Gradle and helps you create website in minutes. 

$ curl http://glide-gae.appspot.com/install | sh
$ mkdir hello-glide
$ cd hello-glide
$ echo "hello glide" > hello.html
 Download

Gaelyk template is also available for download directly from BinTray repository. Download the template and extract it.

Download Template

Run Server

Lazybones

Projects created by Lazybones are using Gradle as a build tool. When running server locally changes in static resources, templates and groovlets are synchronized immediately. 

$ ./gradlew {{runCommand}}

Go to http://localhost:8080 in your browser.

 Glide

Run glide command in project directory. 

$ glide

Go to http://localhost:8080 in your browser.

Deploy into Cloud

Lazybones

If you have already signed up for Google App Engine and set application identifier in the first step you can deploy right away. 

$ ./gradlew {{deployCommand}}

Go to http://yourappid.appspot.com/ in your browser.

 Glide

Sign up for Google App Engine and create new application. Create __glide.groovy file in your project with following content. 

app {
    name = "yourappid"
    version = "1"
}

Run following command:

$ glide deploy

Go to http://yourappid.appspot.com/ in your browser.

 Download
